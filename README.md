# 오늘의 배우운세 MVP

오늘의 운세처럼 오늘 할 배우 연습 활동 하나를 가볍게 뽑아주는 모바일 웹 MVP입니다.

배포 URL: https://acttub.github.io/day/

## 핵심 방향

- 첫 화면에서 바로 이해되어야 합니다: **오늘 할 연습 활동 하나를 뽑는 서비스**.
- 복잡한 상태 선택, 진단, 분석, 영상 업로드 없이 버튼 하나로 시작합니다.
- 결과는 `오늘의 운세 카드`처럼 짧게 보여주고, 마음에 들면 10분 실행 모드로 이어집니다.
- 대본/장면/평가가 아니라 배우훈련 활동 추천만 다룹니다.

## 사용자 흐름

```txt
접속
→ 오늘 활동 뽑기
→ 오늘의 연습 활동 카드
→ 10분 시작하기
→ 1/4~4/4 단계 진행
→ 30초 기록
```

## 화면 구성

### 첫 화면

- 오늘의 배우운세
- 오늘 할 연습 활동 하나만 뽑아드릴게요.
- 오늘 활동 뽑기

### 결과 카드

- 오늘의 연습 활동 제목
- 운세처럼 읽히는 한 줄 설명
- 10분 시작하기 CTA
- 오늘 할 일 4단계
- 운세 한 줄
- 공유/기록 버튼

### 실행 모드

- 0~2분 / 2~5분 / 5~8분 / 8~10분
- 한 화면에 한 지시문
- 이전/다음 단계

### 기록

- 해봤는지
- 내일도 뽑을지
- 오늘 남은 한 문장

```bash

```

현재 테스트는 다음을 확인합니다.

- 배우훈련-only 가드레일
- 10분 실행 모드 단계 상태
- 금지 표현 미노출
