# 오늘의 배우훈련 MVP — 구현본

매일 하나씩 달라지는 10분 배우훈련 추천 웹 MVP입니다.

## 핵심 방향

- 사용자가 입력하지 않아도 오늘의 배우훈련 하나를 랜덤 추천합니다.
- 같은 날짜에는 같은 훈련이 나오고, 날짜가 바뀌면 다른 훈련이 나옵니다.
- 추천 대상은 장면 분석이나 대본 연습이 아니라 **배우훈련 자체**입니다.
- 훈련 카테고리는 뷰포인트 영감, 마이즈너 영감, 즉흥/상상, 감각/관찰, 호흡/집중입니다.
- 결과는 오늘 10분간 실제로 무엇을 해야 하는지 자세히 보여줍니다.

## 기능

- 랜딩 화면
- 매일 랜덤 배우훈련 추천
- 20개 이상 훈련 라이브러리
- 10분 진행법 상세 안내
- 오늘 붙잡을 질문
- 할 때 기억할 것
- 끝나고 체크
- 결과 복사
- 공유 문구 복사
- 1분 기록 저장
- 최근 추천 훈련 로컬 저장
- 기본 이벤트 로컬 기록

## 실행

```bash
npm test
npm start
```

브라우저에서 열기:

```txt
http://localhost:4173
```

## 테스트

```bash
npm test
```

테스트 범위:

- 같은 날짜에는 같은 추천, 다른 날짜에는 다른 추천
- 훈련 라이브러리 20개 이상
- 배우훈련 카테고리만 포함
- 장면/영상/대본 분석 문구 미노출
- 각 훈련의 10분 진행법 상세성
- 평가/점수/강점/약점 표현 금지
- 공유 문구 개인정보 최소화

## 현재 구현 방식

외부 API 없이 로컬 훈련 라이브러리에서 날짜 기반 랜덤 추천을 합니다. 따라서 바로 테스트하거나 GitHub Pages로 배포해 사용할 수 있습니다.

나중에 개인화 추천을 붙일 때도 핵심은 유지합니다: **배우훈련만 추천한다.**
