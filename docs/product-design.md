# 오늘의 배우훈련 제품 설계

## 1. 결론

`오늘의 배우훈련`은 추천 서비스가 아니라 **배우훈련 실행 가이드 서비스**로 정의한다.

사용자가 얻는 핵심 가치는 `오늘 뭘 연습하지?`라는 막힘을 없애고, 어려운 훈련명을 10분 안에 따라 할 수 있는 구체적 행동으로 번역해주는 것이다.

## 2. 첫 타깃 사용자

1차 타깃은 `혼자 연습하는 배우 지망생/초보 배우`다.

특징:
- 연습 의지는 있지만 오늘 뭘 해야 할지 자주 막힌다.
- 전문 훈련명은 들어봤지만 정확히 어떻게 하는지 모른다.
- 대본, 상대, 연습실이 없어도 할 수 있는 짧은 루틴이 필요하다.
- 긴 강의보다 지금 바로 따라 할 수 있는 안내를 원한다.

## 3. 한 줄 약속

> 어려운 배우훈련을 오늘 바로 따라 할 수 있는 10분 루틴으로 바꿔드립니다.

## 4. MVP 사용자 흐름

```txt
방문
→ 오늘 상태 선택
→ 오늘 훈련 받기
→ 훈련 이해
→ 10분 진행
→ 끝나고 30초 기록
→ 내일 재방문
```

## 5. 화면별 역할

### 랜딩

목표: 사용자가 서비스의 약속을 5초 안에 이해하고 시작하게 한다.

핵심 문구:
- 오늘 뭘 연습할지 고민하지 마세요.
- 어려운 배우훈련을 오늘 바로 따라 할 수 있는 10분 루틴으로 바꿔드립니다.

선택지:
- 몸이 굳어 있음
- 집중이 흩어짐
- 말이 급해짐
- 상상이 막힘
- 반응이 둔함
- 그냥 오늘 하나

### 결과

목표: 추천명만 던지지 않고, 훈련을 이해하고 바로 실행하게 한다.

필수 섹션:
1. 오늘의 배우훈련
2. 이 훈련은 뭐예요?
3. 언제 하면 좋아요?
4. 오늘 이걸 하는 이유
5. 10분 진행법
6. 잘하려고 하지 말고 볼 것
7. 오늘 붙잡을 질문
8. 흔히 헷갈리는 점
9. 할 때 기억할 것
10. 끝나고 체크
11. 안전하게 하기

### 기록

목표: 사용자가 훈련 후 감각 하나를 남기게 해서 재방문 이유를 만든다.

질문:
- 실제로 10분을 해봤나요?
- 끝나고 무엇이 가장 남았나요?
- 내일도 받고 싶나요?
- 오늘 남은 한 문장

## 6. 추천 로직

초기 MVP는 외부 AI 없이 다음 방식으로 작동한다.

```txt
오늘 날짜 + 오늘 상태 → 카테고리 풀 선택 → 날짜 기반 deterministic 추천
```

예:
- 몸이 굳어 있음 → 뷰포인트 영감 / 감각·관찰
- 집중이 흩어짐 → 호흡·집중 / 감각·관찰
- 말이 급해짐 → 호흡·집중 / 마이즈너 영감
- 상상이 막힘 → 즉흥·상상 / 감각·관찰
- 반응이 둔함 → 마이즈너 영감 / 뷰포인트 영감

## 7. 지금 하지 않을 것

명확히 제외한다.

- 장면 분석
- 대본/대사 기반 연습
- 영상 업로드
- acttub 질문 대화 CTA
- 점수/등급/강점/약점 표현
- AI가 사용자를 평가하는 문장
- 정식 교육을 대체한다는 주장

## 8. 활성화 지표

MVP에서 가장 중요한 지표는 가입이 아니라 **실행 완료**다.

1차 핵심 지표:
- 방문자 중 `오늘 훈련 받기` 클릭률
- 결과 화면에서 30초 이상 머문 비율
- `훈련 해봤다` 클릭률
- 30초 기록 저장률
- 다음날 재방문률

정성 지표:
- 사용자가 훈련을 이해했다고 말하는가?
- 10분 동안 실제로 했는가?
- 끝나고 남은 감각을 말할 수 있는가?
- 다음날 다시 받을 의사가 있는가?

## 9. 1주 검증 계획

### 목표

5명의 배우 지망생/초보 배우에게 실제 링크를 보내고, `훈련 이해 → 실행 → 기록`이 되는지 확인한다.

### 프로토콜

1. 링크 전달
2. 사용자가 상태 하나 선택
3. 오늘 훈련 받기
4. 설명 읽고 10분 실행
5. 30초 기록 작성
6. 5분 후속 질문

### 후속 질문

- 훈련이 뭔지 이해됐나요?
- 10분 동안 실제로 할 수 있었나요?
- 설명 중 가장 도움이 된 섹션은 무엇인가요?
- 헷갈린 문장이 있었나요?
- 내일도 하나 더 받고 싶나요?
- 이걸 돈 내고 쓴다면 어떤 조건이어야 하나요?

### 통과 기준

5명 중 3명 이상이:
- 설명만 보고 10분 실행
- 끝나고 한 문장 기록
- 다음날 재사용 의사 표현

## 10. 다음 제품 단계

### V1

- 훈련 50개로 확장
- 기록 히스토리 화면
- 7일 연속 루틴
- 카카오/인스타 공유용 카드

### V2

- 사용자의 기록 기반 추천
- 훈련 난이도 조절
- 초보/입시/전공/프로 모드 분리
- 훈련 세트: 3일 집중, 7일 감각 깨우기 등

### 유료화 가설

무료:
- 오늘 훈련 1개
- 최근 기록 로컬 저장

유료 후보:
- 7일/14일 훈련 코스
- 오디션 전 루틴팩
- 수업 전 워밍업팩
- 개인 기록 기반 추천
- 선생님/스터디 그룹용 훈련 링크 발송

## 11. 팀 실행 분담

PM:
- 배우 5명 검증 모집
- 후속 인터뷰
- 훈련 이해도 문장 개선

Frontend/AI:
- 상태 기반 추천 UX
- 기록 저장/히스토리
- 공유 카드

Backend/Ops:
- 정적 배포 안정화
- 간단 로그 수집 경로 검토
- 향후 계정/기록 저장 구조 검토

## 12. 현재 구현 상태

이번 버전은 아래까지 구현한다.

- 오늘 상태 선택
- 상태 기반 훈련 추천
- 배우훈련 전용 카테고리
- 상세 훈련 설명
- 10분 진행법
- 잘하려고 하지 말고 볼 것
- 흔히 헷갈리는 점
- 안전 안내
- 30초 기록
- 테스트 기반 금지 문구 검증
