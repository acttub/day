# 오늘의 연기 훈련 MVP — 구현본

매일 하나씩 달라지는 10분 배우훈련 추천 웹 MVP입니다.

## 핵심 방향

- 사용자가 입력하지 않아도 오늘의 훈련 하나를 랜덤 추천합니다.
- 같은 날짜에는 같은 훈련이 나오고, 날짜가 바뀌면 다른 훈련이 나옵니다.
- 훈련은 장면 기반일 수도 있고, 뷰포인트 영감, 마이즈너 영감, 감각/관찰, 호흡/집중 워밍업일 수도 있습니다.
- 결과는 “무엇을 왜 하는지”보다 “지금 10분간 어떻게 할지”를 자세히 보여줍니다.

## 기능

- 랜딩 화면
- 매일 랜덤 훈련 추천
- 20개 훈련 라이브러리
- 10분 진행법 상세 안내
- 오늘 붙잡을 질문
- 할 때 기억할 것
- 끝나고 체크
- 결과 복사
- 공유 문구 복사
- 1분 기록 저장
- 최근 추천 훈련 로컬 저장
- 기본 이벤트 로컬 기록

```bash

```

테스트 범위:

- 같은 날짜에는 같은 추천, 다른 날짜에는 다른 추천
- 훈련 라이브러리 20개 이상
- 각 훈련의 10분 진행법 상세성
- 장면 진단/피드백형 문구 미노출
- 평가/점수/강점/약점 표현 금지
- 공유 문구 개인정보 최소화

## 현재 구현 방식

외부 API 없이 로컬 훈련 라이브러리에서 날짜 기반 랜덤 추천을 합니다. 따라서 최우영 혼자 바로 테스트하거나 GitHub에서 clone해 실행할 수 있습니다.

나중에 개인화 추천을 붙일 때는 `src/trainingGenerator.js`의 `generateTraining()`을 사용자 기록/컨디션 기반 추천으로 확장하면 됩니다.
