# 오늘의 배우훈련 MVP — 구현본

매일 하나씩 달라지는 10분 배우훈련 추천 웹 MVP입니다.

배포 URL: https://acttub.github.io/day/

## 핵심 방향

- 추천 서비스가 아니라 **배우훈련 실행 가이드 서비스**입니다.
- 사용자가 입력하지 않아도 시작할 수 있고, 오늘 상태를 고르면 더 맞는 카테고리에서 추천합니다.
- 추천 대상은 장면 분석이나 대본 연습이 아니라 **배우훈련 자체**입니다.
- 훈련 카테고리는 뷰포인트 영감, 마이즈너 영감, 즉흥/상상, 감각/관찰, 호흡/집중입니다.
- 결과는 오늘 10분간 실제로 무엇을 해야 하는지 자세히 보여줍니다.

## 기능

- 모바일 우선 랜딩 화면
- 오늘 상태 선택
- 상태 기반 배우훈련 추천
- 날짜 기반 deterministic 추천
- 20개 이상 훈련 라이브러리
- 훈련 이해 설명
- 언제 하면 좋은지 안내
- 10분 진행법 상세 안내
- 잘하려고 하지 말고 볼 것
- 흔히 헷갈리는 점
- 오늘 붙잡을 질문
- 끝나고 체크
- 안전 안내
- 전체 가이드 복사
- 공유 문구 복사
- 30초 기록 저장
- 최근 추천 훈련 로컬 저장
- 기본 이벤트 로컬 기록

## 실행

```bash
npm test
npm start
```

브라우저에서 열기:

```txt
http://localhost:4173
```

## 테스트

```bash
npm test
```

테스트 범위:

- 같은 날짜와 같은 상태에는 같은 추천
- 상태 선택지는 배우훈련 카테고리로만 연결
- 훈련 라이브러리 20개 이상
- 배우훈련 카테고리만 포함
- 장면/영상/대본 분석 문구 미노출
- 각 훈련의 10분 진행법 상세성
- 각 훈련의 이해 가이드 포함
- 평가/점수/강점/약점 표현 금지
- 공유 문구 개인정보 최소화

## 현재 구현 방식

외부 API 없이 로컬 훈련 라이브러리에서 날짜와 오늘 상태 기반 추천을 합니다. 따라서 바로 테스트하거나 GitHub Pages로 배포해 사용할 수 있습니다.

나중에 개인화 추천을 붙일 때도 핵심은 유지합니다: **배우훈련만 추천하고, 바로 실행할 수 있게 설명한다.**
